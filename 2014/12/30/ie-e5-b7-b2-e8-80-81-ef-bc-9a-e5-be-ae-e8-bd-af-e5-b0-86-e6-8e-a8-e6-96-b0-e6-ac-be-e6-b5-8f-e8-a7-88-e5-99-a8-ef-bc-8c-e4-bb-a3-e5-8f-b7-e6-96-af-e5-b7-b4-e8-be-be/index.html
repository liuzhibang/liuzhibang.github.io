IE 已老：微软将推新款浏览器，代号 “斯巴达”

一提起微软的浏览器——OMG，IE？

也许连微软也意识到，这个逐渐沦为“用于下载 Chrome 或 Firefox 的浏览器”，已成为来年最重头戏产品 Windows 10 当中一大软肋。据 ZDNet 报道，微软即将推出一款全新的浏览器。

一款代号为 “Spartan” 的新款浏览器有望随 Windows 10 发布。Thomas Nigro 是著名播放器 VLC 的开发者，同时也是 MSP 项目成员之一，早前在 Twitter 上就透露他曾听说微软准备推出一款全新的浏览器。

据称全新的微软浏览器，将在使用体验上更接近对手 Chrome 和 Firefox ，不仅相比 IE 体积上十分轻盈，而且更支持 extension 插件。当然出于兼容性考虑，“Spartan”依然会基于微软的 Chakra JavaScript engine 和 Trident rendering engine 开发（非 WebKit）。

但 ZDNet 作者 Mary Jo Foley 从知情人士得知，这款浏览器将不叫 IE12，甚至有可能根本不会沿用 Internet Explorer 这个品牌。当然目前断定微软在下一代浏览器中采用全新品牌还言之尚早，但也有人提到早在今年初 IE 开发团队在一个 Reddit 在线访谈中就提过微软内部正在积极考虑更换这个已使用了 19 年的品牌，因为 IE 的名声已经烂透。

最后有两点情报值得关注。

1. 无论是否取得成功，Spartan 无疑将代表微软新一代浏览器的发展方向。但在一夜之间放弃传统架构的 IE 并不现实，基于兼容性的考量，在 Windows 10 中 Spartan 将会和 IE11 并存很长一段时间。

2. 有消息称 Spartan 将会在来年 1 月 21 日的 Windows 10 发布会上亮相，不过根据目前 Spartan 的开发进度，能否如期展示依然成谜。

2.1. Spartan 有可能登陆如 iOS、Android 等非 Windows 平台吗？早前 IE 团队表示没有将 IE 移植到其他平台上的打算。但，他们说的是 “IE”。
